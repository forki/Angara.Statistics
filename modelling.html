Modelling

This tutorial gives you an introduction to using the package for Bayesian modelling. It uses the same data and model as an (OpenBUGS tutorial)[http://www.openbugs.net/Manuals/Tutorial.html].

This example is taken from Table 3 of Crowder (1978) and concerns the proportion of seeds that germinated on each of 21 plates arranged according to a 2 by 2 factorial layout by seed and type of root extract. The data are shown below, where ri and ni are the number of germinated and the total number of seeds on the i-th plate, i = 1, ..., N. These data are also analysed by, for example, Breslow and Clayton (1993).

The model is essentially a random effects logistic regression, allowing for over-dispersion. If p_i is the probability of germination on the i-th plate, we assume

ri ~ Binomial(ni, p_i)

logit(pi) = a0 + a1 x1i + a2 x2i + a12 x1i x2i + bi

b_i ~ Normal(0, tau)

where x1i and x2i are the seed type and root extract of the i-th plate, and an interaction term a12 x1i x2i is included.

let log_likelihood (pp:Parameters) =
    let a0 = pp.GetValue("a0")
    let a1 = pp.GetValue("a1")
    let a2 = pp.GetValue("a2")
    let a12 = pp.GetValue("a12")
    let sigma = pp.GetValue("sigma")
    let b = pp.GetValues("b")
    let logistic x =
        let ex = exp x
        if ex>maxint then 1. elif ex = -infinity then 0. else ex/(1.+ex)
    let mutable logl = 0.
    for i in 0..N-1 do
        let logit_p = a0 + a1*x1.[i] + a2*x2.[i] + a12*x1.[i]*x2.[i] + b.[i]
        let p = logistic(logit_p)
        let logl_r = log_pdf (Binomial(n.[i], p)) r.[i]
        let logl_b = log_pdf (Normal(0., sigma)) b.[i]
        logl <- logl + logl_r + logl_b
    logl // return the resulting sum
